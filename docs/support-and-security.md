# Tracey Support and Security Policy

## Support

Support email: eugenktheba@gmail.com

Support level: Best-effort support by email.

Tracey is currently a free beta app. No formal response-time service level agreement is provided.

When reporting an issue, include:

- Jira Cloud site URL, if it can be shared.
- Project key.
- Tracey view where the issue occurs.
- Screenshot or screen recording.
- Expected behavior.
- Actual behavior.
- Browser and operating system.
- Approximate time the issue happened.

## Security Contact

Security issues should be reported to eugenktheba@gmail.com.

Please include:

- A clear description of the issue.
- Reproduction steps.
- Potential impact.
- Whether the issue has been disclosed publicly.

## Data Handling Summary

Tracey runs on Atlassian Forge and uses Atlassian-hosted app execution and storage.

Tracey does not intentionally send Jira issue data to external analytics or advertising services.

## Responsible Disclosure

Please do not publicly disclose a security issue before Eugene K has had reasonable time to investigate and respond.
