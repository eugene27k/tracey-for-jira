Help

Components View

Rows are Jira issues. Columns are affected components from Tracey's structured field. Green cells mean the issue is linked to that component.

Story-to-Story View

Rows and columns are stories. Cells show standard Jira issue links: red for blocks, purple for duplicates, yellow for clones, and grey for relates. Half-filled cells indicate the relation points from the column issue to the row issue.

Hierarchical View

Cards show issue hierarchy. Drag eligible leaf issues onto valid parent cards to assign hierarchy. Parent Coverage excludes epics and shows how many non-epic issues have a parent.

Filters

Filters limit the Jira issues loaded into the current view. Use Refresh after changing filters or when Jira data has changed.
